# DOffice Front-end

This is an [Expo](https://expo.dev) project for the DOfice front-end part created with [`create-expo-app`](https://www.npmjs.com/package/create-expo-app).

## Setting up the project

1. Install dependencies

   ```bash
   npm install
   ```

2. Start the app

   ```bash
    npx expo start --tunnel
   ```
